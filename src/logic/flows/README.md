# Flows

Flows sisaldab Cyber32 süsteemi töövooge.

Flow ühendab mitu reeglit, tingimust ja tegevust üheks tervikuks protsessiks.

## Vastutus

- Töövoogude kirjeldamine
- Töövoogude käivitamine
- Töövoogude peatamine
- Töövoogude oleku jälgimine

## Näited

Planting Flow

Navigation Flow

Obstacle Avoidance Flow

Battery Protection Flow

Home Automation Flow

## Flow struktuur

Trigger
↓
Condition
↓
Action
↓
Next Step

## Flows ei vastuta

- Riistvara juhtimise eest
- Driverite töö eest
- Plug-and-play eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Võimaldada keerukate protsesside loomist ilma programmeerimata.
