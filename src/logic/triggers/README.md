# Triggers

Triggers sisaldab Cyber32 süsteemi sündmusi, mis käivitavad loogika.

Trigger on Flow alguspunkt.

## Vastutus

- Sündmuste jälgimine
- Loogika käivitamine
- Triggerite registreerimine
- Triggerite filtreerimine

## Näited

Position Changed

Battery Level Changed

Temperature Changed

Button Pressed

Module Connected

Module Disconnected

System Started

## Trigger struktuur

Event
↓
Trigger
↓
Flow

## Triggers ei vastuta

- Tingimuste hindamise eest
- Tegevuste käivitamise eest
- Riistvara juhtimise eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele Logic komponentidele.

## Eesmärk

Määrata, millal Cyber32 loogika peab käivituma.
