# Actions

Actions sisaldab Cyber32 süsteemi tegevusi, mida loogika saab käivitada.

Action on konkreetne tegevus, mis toimub pärast Triggeri ja Conditioni edukat täitmist.

## Vastutus

- Tegevuste kirjeldamine
- Tegevuste käivitamine
- Tegevuste oleku jälgimine
- Tegevuste tulemuste raporteerimine

## Näited

Start Motor

Stop Motor

Set Relay

Save Data

Send Notification

Start Flow

Stop Flow

Update Display

## Action struktuur

Trigger
↓
Condition
↓
Action

## Actions ei vastuta

- Tingimuste hindamise eest
- Triggerite jälgimise eest
- Riistvara madala taseme juhtimise eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele Logic komponentidele.

## Eesmärk

Võimaldada Cyber32 süsteemil teha reaalseid tegevusi vastusena sündmustele ja tingimustele.
