# Variables

Variables sisaldab Cyber32 süsteemi muutujaid.

Variable võimaldab loogikal salvestada, lugeda ja jagada väärtusi erinevate Flow-de, Triggerite, Conditionite ja Actionite vahel.

## Vastutus

- Väärtuste hoidmine
- Väärtuste uuendamine
- Väärtuste jagamine komponentide vahel
- Muutujate tüübi haldamine

## Näited

Battery Level

Current Position

Target Position

Plant Counter

System Mode

Temperature Threshold

## Võimalikud tüübid

- Boolean
- Integer
- Float
- String
- Array
- Object

## Variable struktuur

Name
↓
Type
↓
Value

## Variables ei vastuta

- Triggerite käivitamise eest
- Tingimuste hindamise eest
- Tegevuste teostamise eest
- Riistvara juhtimise eest

Need ülesanded kuuluvad teistele Logic komponentidele.

## Eesmärk

Võimaldada Cyber32 loogikal säilitada ja kasutada andmeid sõltumata konkreetsest riistvarast.
