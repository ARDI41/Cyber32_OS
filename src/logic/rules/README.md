# Rules

Rules sisaldab Cyber32 süsteemi tingimusreegleid.

Reegel kirjeldab, millal midagi peab juhtuma.

## Vastutus

- Tingimuste kirjeldamine
- Reeglite hindamine
- Reeglite aktiveerimine
- Reeglite prioriteetide haldamine

## Näited

IF Battery Level < 20%

IF Temperature > 50°C

IF Position Changed

IF Distance < 10cm

## Reegli struktuur

Condition
↓
Evaluation
↓
Result

## Rules ei vastuta

- Tegevuste teostamise eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele Logic komponentidele.

## Eesmärk

Kirjeldada, millal süsteem peab reageerima.
