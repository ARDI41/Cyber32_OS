# Scheduler

Scheduler vastutab Cyber32 süsteemi ülesannete käivitamise ajastamise eest.

Scheduler otsustab millal ja millises järjekorras süsteemi komponendid töötavad.

## Vastutus

- Ülesannete ajastamine
- Perioodiliste ülesannete käivitamine
- Prioriteetide haldamine
- Ressursside efektiivne kasutamine

## Näited

- Andurite perioodiline lugemine
- Telemeetria saatmine
- Moodulite tervisekontroll
- Dashboardi uuendamine

## Scheduler ei vastuta

- Ülesannete sisu eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Tagada, et Cyber32 süsteem töötaks sujuvalt ka siis, kui korraga on aktiivsed kümned või sajad komponendid.
