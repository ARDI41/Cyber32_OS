# Runtime

Runtime vastutab Cyber32 süsteemi töötavate komponentide elutsükli haldamise eest.

Runtime ühendab registreeritud komponendid töötavaks süsteemiks.

## Vastutus

- Komponentide käivitamine
- Komponentide peatamine
- Komponentide elutsükli haldamine
- Ressursside jälgimine
- Süsteemi töö koordineerimine

## Alamkomponendid

- Scheduler
- Tasks
- State Machine
- Resources
- Lifecycle

## Runtime eesmärk

Tagada, et kõik Cyber32 komponendid töötaksid kontrollitud ja ennustataval viisil.

## Runtime ei vastuta

- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
