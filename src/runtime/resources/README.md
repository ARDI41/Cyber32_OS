# Resources

Resources vastutab Cyber32 süsteemi ressursside jälgimise ja haldamise eest.

Resource on piiratud süsteemi ressurss, mida kasutavad komponendid oma tööks.

## Vastutus

- Ressursside jälgimine
- Ressursside reserveerimine
- Ressursside vabastamine
- Ressursikonfliktide vältimine
- Ressursikasutuse statistika kogumine

## Näited

- CPU aeg
- Mälu
- Salvestusruum
- Võrguühendus
- CAN siin
- I2C siin
- UART port

## Resources ei vastuta

- Ressursside füüsilise juhtimise eest
- Riistvara draiverite eest
- Plug-and-play eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Tagada, et Cyber32 süsteemi komponendid kasutaksid ressursse kontrollitult ja ilma konfliktideta.
