# Motor Devices

Motor Devices sisaldab Cyber32 süsteemi mootoriseadmeid.

Device ühendab ühe või mitu mootoriga seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Mootori oleku haldamine
- Mootori võimekuste kirjeldamine
- Mootori juhtimise lihtsustamine
- Ühtse liidese pakkumine süsteemile

## Näited

- Servo Device
- DC Motor Device
- Stepper Device
- BLDC Device
- Linear Actuator Device

## Device ei vastuta

- Madala taseme PWM juhtimise eest
- Riistvaraspetsiifiliste detailide eest
- Navigatsiooni eest
- Automaatika eest

Need ülesanded kuuluvad teistele kihtidele.
