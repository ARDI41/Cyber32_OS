# Display Devices

Display Devices sisaldab Cyber32 süsteemi kuvaseadmeid.

Device ühendab ühe või mitu ekraaniga seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Kuvaseadme oleku haldamine
- Kuvamisvõimekuste kirjeldamine
- Ühtse liidese pakkumine süsteemile
- Kuvaseadme kasutamise lihtsustamine

## Näited

- OLED Device
- LCD Device
- TFT Device
- E-Paper Device
- LED Matrix Device

## Device ei vastuta

- Madala taseme SPI või I2C suhtluse eest
- Graafikate renderdamise eest
- Dashboardi loogika eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele kihtidele.
