# Communication Devices

Communication Devices sisaldab Cyber32 süsteemi side- ja võrguseadmeid.

Device ühendab ühe või mitu sidega seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Sideühenduse oleku haldamine
- Ühendusvõimekuste kirjeldamine
- Ühtse liidese pakkumine süsteemile
- Sidevahendite kasutamise lihtsustamine

## Näited

- WiFi Device
- Bluetooth Device
- LoRa Device
- CAN Device
- Ethernet Device
- RS485 Device

## Device ei vastuta

- Madala taseme UART, SPI või CAN suhtluse eest
- Võrgupoliitikate eest
- Automaatika eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele kihtidele.
