# Power Devices

Power Devices sisaldab Cyber32 süsteemi toitega seotud seadmeid.

Device ühendab ühe või mitu toitehaldusega seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Toiteallika oleku haldamine
- Akuandmete koondamine
- Energiaga seotud võimekuste kirjeldamine
- Ühtse liidese pakkumine süsteemile

## Näited

- Battery Device
- Charger Device
- BMS Device
- Power Supply Device
- Power Monitor Device

## Device ei vastuta

- Madala taseme ADC lugemise eest
- Madala taseme riistvara juhtimise eest
- Energiapoliitika eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele kihtidele.
