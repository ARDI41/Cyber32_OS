# Actuator Devices

Actuator Devices sisaldab Cyber32 süsteemi täiturseadmeid.

Device ühendab ühe või mitu täiturseadmega seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Täiturseadme oleku haldamine
- Täiturseadme võimekuste kirjeldamine
- Ühtse liidese pakkumine süsteemile
- Täiturseadme töö lihtsustamine

## Näited

- Relay Device
- Solenoid Device
- Valve Device
- Buzzer Device
- Lock Device

## Device ei vastuta

- Madala taseme GPIO või PWM juhtimise eest
- Riistvaraspetsiifiliste detailide eest
- Automaatika eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele kihtidele.
