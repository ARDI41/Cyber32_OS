# Storage Devices

Storage Devices sisaldab Cyber32 süsteemi salvestusseadmeid.

Device ühendab ühe või mitu salvestusega seotud draiverit terviklikuks kasutatavaks seadmeks.

## Vastutus

- Salvestusseadme oleku haldamine
- Salvestusvõimekuste kirjeldamine
- Ühtse liidese pakkumine süsteemile
- Salvestusseadme kasutamise lihtsustamine

## Näited

- Flash Device
- EEPROM Device
- FRAM Device
- SD Card Device
- External Memory Device

## Device ei vastuta

- Madala taseme lugemise ja kirjutamise eest
- Failisüsteemi poliitikate eest
- Konfiguratsioonide haldamise eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele kihtidele.
