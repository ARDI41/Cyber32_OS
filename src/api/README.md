# API

API võimaldab Cyber32 komponentidel, rakendustel ja välissüsteemidel süsteemiga suhelda.

API on ametlik ligipääsupunkt Cyber32 funktsioonidele.

## Vastutus

- Andmete pakkumine
- Käskude vastuvõtmine
- Süsteemi oleku jagamine
- Teenuste avalikustamine
- Integratsioonide toetamine

## Alamkomponendid

- Internal
- External
- WebSocket
- REST

## Näited

- Moodulite nimekiri
- Device olekud
- Telemeetria andmed
- Loogika juhtimine
- Dashboardi päringud

## API ei vastuta

- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest
- Kasutajaliidese renderdamise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
