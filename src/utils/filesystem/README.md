# Filesystem

Filesystem sisaldab Cyber32 failisüsteemiga seotud abifunktsioone.

## Vastutus

- Failiteede töötlemine
- Failide olemasolu kontrollimine
- Failinimede töötlemine
- Kataloogide töötlemine
- Failisüsteemi abifunktsioonide pakkumine

## Näited

- File Exists
- Directory Exists
- Path Normalize
- Get Extension
- Get Filename
- Create Directory

## Võimalikud kasutuskohad

- Storage Manager
- OTA Manager
- Logging
- Dashboard
- Config süsteem

## Eesmärk

Pakkuda kogu Cyber32 süsteemile ühtset failisüsteemi tööriistakomplekti.

## Filesystem ei vastuta

- Andmete salvestamise eest
- Salvestusseadmete juhtimise eest
- Riistvara juhtimise eest
- Rakendusloogika eest
