# Strings

Strings sisaldab Cyber32 tekstide ja stringide töötlemise abifunktsioone.

## Vastutus

- Stringide teisendamine
- Stringide võrdlemine
- Stringide valideerimine
- Stringide vormindamine
- Stringide parsimine

## Näited

- To Upper Case
- To Lower Case
- Trim
- Split
- Join
- Replace
- Parse Number

## Eesmärk

Pakkuda kogu Cyber32 süsteemile ühtset tekstide töötlemise tööriistakomplekti.

## Strings ei vastuta

- Failide lugemise eest
- Võrguühenduste eest
- Riistvara juhtimise eest
- Rakendusloogika eest
