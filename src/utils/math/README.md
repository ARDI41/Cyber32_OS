# Math

Math sisaldab Cyber32 matemaatika ja arvutuste abifunktsioone.

## Vastutus

- Matemaatiliste arvutuste toetamine
- Geomeetriliste arvutuste toetamine
- Statistiliste arvutuste toetamine
- Ühikute teisendamine
- Üldiste matemaatikafunktsioonide pakkumine

## Näited

- Distance Calculation
- Angle Calculation
- Average
- Min / Max
- Clamp
- Map Range
- Unit Conversion

## Võimalikud kasutuskohad

- Navigatsioon
- Robootika
- Telemeetria
- Sensorite andmetöötlus
- Automaatika

## Eesmärk

Pakkuda kogu Cyber32 süsteemile ühtset matemaatiliste tööriistade kogumit.

## Math ei vastuta

- Sensorite lugemise eest
- Riistvara juhtimise eest
- Loogika otsuste eest
- Rakendusloogika eest
