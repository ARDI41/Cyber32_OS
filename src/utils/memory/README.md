# Memory

Memory sisaldab Cyber32 mäluhalduse abifunktsioone.

## Vastutus

- Mälu kasutuse jälgimine
- Mälu statistika kogumine
- Mälu abifunktsioonid
- Mälu valideerimine
- Mälu diagnostika toetamine

## Näited

- Free Memory
- Used Memory
- Memory Statistics
- Memory Validation
- Memory Debugging

## Eesmärk

Pakkuda kogu Cyber32 süsteemile ühtset mäluhalduse tööriistakomplekti.

## Memory ei vastuta

- Mälu füüsilise eraldamise eest operatsioonisüsteemi tasemel
- Storage seadmete eest
- Failisüsteemi eest
- Rakendusloogika eest
