# Utils

Utils sisaldab Cyber32 süsteemi üldotstarbelisi abifunktsioone.

Need komponendid ei kuulu otseselt ühegi konkreetse süsteemikihi juurde, kuid neid kasutatakse kogu Cyber32 süsteemis.

## Vastutus

- Korduvkasutatavate funktsioonide pakkumine
- Üldiste tööriistade pakkumine
- Süsteemi komponentide toetamine
- Arenduse lihtsustamine

## Alamkomponendid

- Memory
- Strings
- Math
- Time
- Filesystem
- Validation

## Näited

- String teisendused
- Mälu abifunktsioonid
- Matemaatilised arvutused
- Failiteede töötlemine
- Andmete valideerimine
- Ajafunktsioonid

## Utils ei vastuta

- Äriloogika eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Vähendada koodi dubleerimist ja pakkuda kogu süsteemile ühiseid tööriistu.
