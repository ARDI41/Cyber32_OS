# Time

Time sisaldab Cyber32 aja ja kuupäevaga seotud abifunktsioone.

## Vastutus

- Aja teisendamine
- Ajavahemike arvutamine
- Ajatemplite töötlemine
- Ajavõrdluste tegemine
- Ajaga seotud abifunktsioonide pakkumine

## Näited

- Timestamp Conversion
- Elapsed Time
- Duration Calculation
- Time Formatting
- Uptime Calculation
- Timeout Helpers

## Võimalikud kasutuskohad

- Telemeetria
- Runtime
- Scheduler
- Logging
- OTA süsteem

## Eesmärk

Pakkuda kogu Cyber32 süsteemile ühtset ajafunktsioonide kogumit.

## Time ei vastuta

- Scheduleri töö eest
- Süsteemikella riistvara eest
- Riistvara juhtimise eest
- Rakendusloogika eest
