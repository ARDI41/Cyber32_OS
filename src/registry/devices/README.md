# Device Registry

Device Registry sisaldab kõiki Cyber32 süsteemis registreeritud Device objekte.

## Vastutus

- Device objektide registreerimine
- Device objektide eemaldamine
- Device objektide otsimine
- Device objektide oleku hoidmine
- Device metadata hoidmine

## Näited

- Servo Device
- GPS Device
- Battery Device
- Relay Device
- Display Device

## Device Registry võimaldab

- Leida seadmeid ID järgi
- Leida seadmeid tüübi järgi
- Leida seadmeid võimekuse järgi
- Jälgida seadmete olekut

## Device Registry ei vastuta

- Device loomise eest
- Plug-and-play eest
- Draiverite töö eest
- Riistvara juhtimise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
