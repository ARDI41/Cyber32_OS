# Service Registry

Service Registry sisaldab kõiki Cyber32 süsteemis registreeritud Service objekte.

## Vastutus

- Service objektide registreerimine
- Service objektide eemaldamine
- Service objektide otsimine
- Service oleku hoidmine
- Service metadata hoidmine

## Näited

- Network Manager
- Storage Manager
- Power Manager
- Telemetry Manager
- OTA Manager
- Security Manager

## Service Registry võimaldab

- Leida teenuseid nime järgi
- Leida teenuseid tüübi järgi
- Kontrollida teenuse olekut
- Leida teenuse pakutavaid võimekusi

## Service Registry ei vastuta

- Teenuse käivitamise eest
- Teenuse töö eest
- Plug-and-play eest
- Riistvara juhtimise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
