# Module Registry

Module Registry sisaldab kõiki Cyber32 süsteemis registreeritud Module objekte.

## Vastutus

- Module objektide registreerimine
- Module objektide eemaldamine
- Module objektide otsimine
- Module oleku hoidmine
- Module metadata hoidmine

## Näited

- GPS Module
- Sensor Module
- Power Module
- Planting Module
- Mower Module

## Module Registry võimaldab

- Leida mooduleid ID järgi
- Leida mooduleid tüübi järgi
- Leida mooduleid võimekuse järgi
- Leida mooduliga seotud Device objektid
- Jälgida moodulite olekut

## Module Registry ei vastuta

- Mooduli loomise eest
- Plug-and-play eest
- Draiverite töö eest
- Riistvara juhtimise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
