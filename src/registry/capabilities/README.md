# Capability Registry

Capability Registry sisaldab kõiki Cyber32 süsteemis registreeritud võimekusi.

Võimekus kirjeldab, mida süsteem, moodul või seade teha oskab.

## Vastutus

- Capability objektide registreerimine
- Capability objektide eemaldamine
- Capability objektide otsimine
- Võimekuste seostamine Device ja Module objektidega

## Näited

- Position
- Speed
- Navigation
- Temperature Measurement
- Distance Measurement
- Relay Switching
- Battery Monitoring
- Video Streaming

## Capability Registry võimaldab

- Leida võimekusi nime järgi
- Leida seadmeid võimekuse järgi
- Leida mooduleid võimekuse järgi
- Kontrollida süsteemi võimalusi

## Capability Registry ei vastuta

- Võimekuste realiseerimise eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
