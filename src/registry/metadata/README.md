# Metadata Registry

Metadata Registry sisaldab Cyber32 süsteemi kirjeldavaid metaandmeid.

Metadata annab süsteemile infot seadmete, moodulite ja teenuste kohta ilma nende sisemist loogikat tundmata.

## Vastutus

- Metadata hoidmine
- Metadata otsimine
- Metadata seostamine objektidega
- Metadata versioonihalduse toetamine

## Näited

- Name
- Description
- Version
- Manufacturer
- Model
- Serial Number
- Supported Protocols
- Supported Capabilities

## Metadata Registry võimaldab

- Kuvada infot kasutajale
- Kontrollida ühilduvust
- Toetada Plug-and-Play süsteemi
- Luua automaatset dokumentatsiooni

## Metadata Registry ei vastuta

- Riistvara juhtimise eest
- Plug-and-play protsessi eest
- Draiverite töö eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
