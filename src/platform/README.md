# Platform

Platform sisaldab riistvaraplatvormi spetsiifilisi teostusi.

Platform võimaldab Cyber32 töötada erinevatel riistvaradel ilma ülejäänud süsteemi muutmata.

## Vastutus

- Platvormispetsiifiliste funktsioonide teostamine
- Riistvaraplatvormide abstraktsioon
- Süsteemi käivitamise toetamine
- Platvormi võimaluste kirjeldamine

## Näited

- ESP32
- ESP32-S3
- STM32
- Raspberry Pi Pico
- Android Host
- Future Cyber32 Board

## Eesmärk

Võimaldada Cyber32 töötamist mitmel erineval platvormil sama arhitektuuri alusel.

## Platform ei vastuta

- Rakendusloogika eest
- Plug-and-play eest
- Dashboardi eest
- Registri haldamise eest
