# Widgets

Widgets sisaldab Dashboardi korduvkasutatavaid kasutajaliidese komponente.

Widget on väikseim iseseisev Dashboardi element.

## Vastutus

- Andmete visualiseerimine
- Kasutaja sisendite vastuvõtmine
- Korduvkasutatavate UI komponentide pakkumine
- Dashboardi ehitusplokkide loomine

## Näited

- Battery Widget
- GPS Widget
- Telemetry Widget
- Camera Widget
- Logic Node Widget
- Module Status Widget

## Widget omadused

- Korduvkasutatav
- Iseseisev
- Konfigureeritav
- Reaalajas uuendatav

## Widgets ei vastuta

- Andmete salvestamise eest
- API loogika eest
- Riistvara juhtimise eest
- Plug-and-play eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
