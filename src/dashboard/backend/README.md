# Dashboard Backend

Dashboard Backend on Dashboardi serveripoolne osa.

Backend vahendab andmeid Dashboardi ja Cyber32 süsteemi vahel.

## Vastutus

- API kasutamine
- Andmete koondamine
- Sessioonide haldamine
- Kasutajate haldamine
- Dashboardi teenuste pakkumine

## Näited

- Moodulite nimekirja laadimine
- Telemeetria päringud
- Loogika salvestamine
- Süsteemi konfiguratsiooni muutmine
- OTA käivitamine

## Backend ei vastuta

- Kasutajaliidese kuvamise eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Driverite töö eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
