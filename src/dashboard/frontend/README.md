# Dashboard Frontend

Dashboard Frontend on Cyber32 kasutajaliidese kliendipoolne osa.

Frontend kuvab kasutajale süsteemi olekut ning võimaldab süsteemiga suhelda.

## Vastutus

- Kasutajaliidese kuvamine
- Kasutaja sisendite vastuvõtmine
- Reaalaja andmete kuvamine
- Loogika koostamise kasutajaliides
- Süsteemi haldusvaadete kuvamine

## Näited

- Moodulite vaade
- Device vaade
- Capability vaade
- Telemeetria vaade
- Logic Builder
- Seadete vaade

## Frontend ei vastuta

- Andmete salvestamise eest
- API äriloogika eest
- Riistvara juhtimise eest
- Plug-and-play eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Seos teiste kihtidega

User
↓
Frontend
↓
Backend
↓
API
↓
Cyber32 System

## Eesmärk

Pakkuda lihtsat ja intuitiivset kasutajaliidest nii algajale kui ka edasijõudnud kasutajale.
