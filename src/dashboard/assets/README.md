# Assets

Assets sisaldab Dashboardi staatilisi ressursse.

Need ressursid ei sisalda äriloogikat, vaid toetavad kasutajaliidest ja kasutajakogemust.

## Vastutus

- Staatiliste failide hoidmine
- Visuaalsete ressursside haldamine
- Teemade toetamine
- Ikonograafia haldamine

## Näited

- Ikoonid
- Logod
- Pildid
- Fondid
- Teemad
- Helifailid
- Animatsioonid

## Assets omadused

- Korduvkasutatavad
- Platvormist sõltumatud
- Versioonitavad
- Optimeeritavad

## Assets ei vastuta

- Andmete töötlemise eest
- API loogika eest
- Riistvara juhtimise eest
- Plug-and-play eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
