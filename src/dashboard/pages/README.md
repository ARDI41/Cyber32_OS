# Pages

Pages sisaldab Dashboardi täisvaateid.

Page ühendab mitu Widget objekti üheks terviklikuks kasutajavaateks.

## Vastutus

- Vaadete koostamine
- Widgetite organiseerimine
- Navigatsiooni toetamine
- Kasutajakogemuse kujundamine

## Näited

- Home Page
- Modules Page
- Devices Page
- Telemetry Page
- Logic Builder Page
- Settings Page
- OTA Page

## Page omadused

- Koosneb Widgetitest
- Võib sisaldada mitut sektsiooni
- Võib kasutada reaalaja andmeid
- Võib sisaldada kasutaja tegevusi

## Pages ei vastuta

- Andmete salvestamise eest
- API loogika eest
- Riistvara juhtimise eest
- Plug-and-play eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
