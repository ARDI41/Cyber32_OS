# Dashboard

Dashboard on Cyber32 süsteemi peamine kasutajaliides.

Dashboard võimaldab kasutajal vaadata süsteemi olekut, hallata mooduleid, luua loogikat ja juhtida Cyber32 süsteemi ilma programmeerimata.

## Vastutus

- Süsteemi visualiseerimine
- Moodulite kuvamine
- Device objektide kuvamine
- Capability objektide kuvamine
- Loogika loomise võimaldamine
- Süsteemi haldamine

## Alamkomponendid

- Backend
- Frontend
- Widgets
- Pages
- Assets

## Põhimõte

Kasutaja näeb:

Modules
↓
Devices
↓
Capabilities

Loogika kasutab:

Capabilities

## Näited

- Moodulite haldus
- Telemeetria kuvamine
- Loogika ehitamine
- Süsteemi konfiguratsioon
- OTA uuendused

## Dashboard ei vastuta

- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest
- Driverite töö eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Võimaldada Cyber32 kasutamist ilma programmeerimiseta.
