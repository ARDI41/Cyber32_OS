# Kernel

Kernel on Cyber32 OS-i keskne juhtkomponent.

Kernel käivitab süsteemi, seob kokku põhikomponendid ning haldab süsteemi elutsüklit.

## Vastutus

* Süsteemi käivitamine
* Süsteemi peatamine
* Komponentide initsialiseerimine
* Süsteemi tööoleku jälgimine
* Põhikomponentide ühendamine

## Kernel ei vastuta

* Seadmete juhtimise eest
* Moodulite tuvastamise eest
* Võrguühenduste eest
* Kasutajaliidese eest

Need ülesanded delegeeritakse teistele süsteemikomponentidele.
