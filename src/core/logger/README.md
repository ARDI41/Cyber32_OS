# Logger

Logger vastutab Cyber32 süsteemi logimise ja diagnostika eest.

## Vastutus

- Süsteemi teadete logimine
- Vigade logimine
- Hoiatuste logimine
- Debug info väljastamine
- Tulevikus logide salvestamine

## Logi tasemed

- DEBUG
- INFO
- WARNING
- ERROR
- CRITICAL

## Logger ei vastuta

- Vigade parandamise eest
- Seadmete juhtimise eest
- Kasutajaliidese kuvamise eest
