# Config

Config vastutab Cyber32 süsteemi seadistuste haldamise eest.

## Vastutus

- Süsteemi konfiguratsioonide hoidmine
- Vaikeväärtuste määramine
- Konfiguratsioonide laadimine
- Konfiguratsioonide salvestamine
- Konfiguratsioonide valideerimine

## Näited

- Seadme nimi
- WiFi seaded
- Moodulite konfiguratsioon
- Dashboardi seaded
- Süsteemi parameetrid

## Config ei vastuta

- Andmete füüsilise salvestamise eest
- Seadmete juhtimise eest
- Süsteemi oleku hoidmise eest
