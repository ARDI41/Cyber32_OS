# System State

System State kirjeldab Cyber32 süsteemi üldist tööolekut.

## Vastutus

- Süsteemi oleku hoidmine
- Olekumuutuste jälgimine
- Olekumuutustest teavitamine

## Süsteemi olekud

- BOOTING
- INITIALIZING
- READY
- WARNING
- ERROR
- UPDATING
- SHUTDOWN

## Näited

BOOTING
→ INITIALIZING
→ READY

READY
→ WARNING

READY
→ ERROR

## System State ei vastuta

- Seadmete oleku hoidmise eest
- Moodulite oleku hoidmise eest
- Aku taseme hoidmise eest
- Võrguühenduste hoidmise eest

Need kuuluvad teiste komponentide vastutusalasse.
