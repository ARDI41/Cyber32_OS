# Registry

Registry on Cyber32 keskne register süsteemis olevate objektide jaoks.

## Vastutus

- Seadmete registreerimine
- Moodulite registreerimine
- Teenuste registreerimine
- Võimekuste registreerimine
- Metaandmete hoidmine

## Registry alamjaotused

- devices
- modules
- services
- capabilities
- metadata

## Näited

Registry võib sisaldada:

- Servo Device
- GPS Device
- Network Service
- OTA Service
- Position Tracking Capability

## Registry ei vastuta

- Seadmete juhtimise eest
- Moodulite tuvastamise eest
- Süsteemi oleku hoidmise eest

Registry ainult hoiab ja haldab registreeritud objekte.
