# PNP

PNP (Plug and Play) vastutab Cyber32 süsteemis moodulite automaatse tuvastamise eest.

Eesmärk on võimaldada kasutajal ühendada moodul ning saada see automaatselt tööle ilma käsitsi seadistamiseta.

## Vastutus

- Moodulite avastamine
- Moodulite identifitseerimine
- Moodulite registreerimine
- Ühilduvuse kontrollimine
- Hot-Plug toe pakkumine

## PNP töövoog

Moodul ühendatakse
↓
Moodul tuvastatakse
↓
Metadata loetakse
↓
Devices registreeritakse
↓
Capabilities registreeritakse
↓
Moodul muutub kasutatavaks

## PNP ei vastuta

- Draiverite töö eest
- Riistvara juhtimise eest
- Rakendusloogika eest
- Kasutajaliidese eest

Need ülesanded kuuluvad teistele kihtidele.
