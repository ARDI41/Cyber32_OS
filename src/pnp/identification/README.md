# Identification

Identification vastutab avastatud mooduli identiteedi ja omaduste tuvastamise eest.

Identification on Plug and Play protsessi teine etapp.

## Vastutus

- Mooduli identiteedi lugemine
- Metadata lugemine
- Versiooni tuvastamine
- Tootja tuvastamine
- Mooduli tüübi määramine

## Metadata näide

Name: GPS Module

Version: 1.0

Manufacturer: Cyber32

Module Type: Navigation

## Näide

Discovery leiab mooduli
↓
Identification loeb metadata
↓
Mooduli tüüp määratakse
↓
Algab registreerimine

## Identification ei vastuta

- Mooduli registreerimise eest
- Ühilduvuse kontrollimise eest
- Device objektide loomise eest

Need ülesanded kuuluvad järgmistele PNP komponentidele.
