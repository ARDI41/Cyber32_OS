# Registration

Registration vastutab tuvastatud mooduli lisamise eest Cyber32 süsteemi.

Registration on Plug and Play protsessi kolmas etapp.

## Vastutus

- Mooduli registreerimine
- Device objektide registreerimine
- Capability objektide registreerimine
- Registry uuendamine
- Süsteemi teavitamine uuest moodulist

## Näide

Discovery leiab mooduli
↓
Identification loeb metadata
↓
Registration loob kirjed registrisse
↓
Moodul muutub süsteemile nähtavaks

## Registreeritavad objektid

- Module
- Devices
- Capabilities
- Metadata

## Registration ei vastuta

- Ühilduvuse kontrollimise eest
- Draiverite töö eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele komponentidele.
