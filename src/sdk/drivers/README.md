# SDK Drivers

SDK Drivers sisaldab juhiseid, malle ja tööriistu uute Cyber32 draiverite loomiseks.

## Vastutus

- Driver mallide pakkumine
- Driver standardite kirjeldamine
- Driver arendamise lihtsustamine
- Ühilduvuse tagamine Cyber32 süsteemiga

## Näited

- Sensor Driver Template
- Motor Driver Template
- Communication Driver Template
- Storage Driver Template

## Driver nõuded

- Kasutab HAL kihti
- Ei sõltu Dashboardist
- Ei sõltu Logic kihist
- Järgib Cyber32 liideseid

## Eesmärk

Võimaldada uute draiverite loomist ühtse standardi järgi.
