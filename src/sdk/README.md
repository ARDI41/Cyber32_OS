# SDK

SDK (Software Development Kit) võimaldab arendajatel luua Cyber32 jaoks uusi draivereid, seadmeid, mooduleid ja laiendusi.

SDK eesmärk on võimaldada süsteemi laiendamist ilma Cyber32 põhituuma muutmata.

## Vastutus

- Arendustööriistade pakkumine
- Näidiste pakkumine
- Mallide pakkumine
- Laienduste loomise lihtsustamine
- Cyber32 standardite järgimise toetamine

## Alamkomponendid

- Drivers
- Devices
- Modules
- Templates
- Tools

## Näited

- Uus Sensor Driver
- Uus Device tüüp
- Uus Module tüüp
- Kohandatud Dashboard Widget
- Kolmanda osapoole laiendus

## SDK ei vastuta

- Süsteemi töö eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Registri haldamise eest

Need ülesanded kuuluvad Cyber32 põhikomponentidele.

## Eesmärk

Võimaldada Cyber32 ökosüsteemi kasvu ilma põhituuma muutmata.
