# SDK Modules

SDK Modules sisaldab juhiseid, malle ja tööriistu uute Cyber32 Module objektide loomiseks.

## Vastutus

- Module mallide pakkumine
- Module standardite kirjeldamine
- Module arendamise lihtsustamine
- Ühilduvuse tagamine Cyber32 süsteemiga

## Näited

- Navigation Module Template
- Sensor Module Template
- Power Module Template
- Custom Module Template

## Module nõuded

- Sisaldab ühte või mitut Device objekti
- Omab Metadata objekti
- Kirjeldab oma Capabilities
- Toetab Plug-and-Play süsteemi
- Järgib Cyber32 liideseid

## Module eesmärk

Ühendada Device objektid kasutajale arusaadavaks funktsionaalseks tervikuks.

## Eesmärk

Võimaldada uute moodulite loomist ühtse Cyber32 standardi järgi.
