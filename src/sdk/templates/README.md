# SDK Templates

SDK Templates sisaldab Cyber32 ametlikke projekti- ja komponendimalle.

Mallid võimaldavad luua uusi Cyber32 komponente kiiresti ja ühtse struktuuriga.

## Vastutus

- Projekti mallide pakkumine
- Driver mallide pakkumine
- Device mallide pakkumine
- Module mallide pakkumine
- Service mallide pakkumine

## Näited

- Basic Driver Template
- Sensor Device Template
- Navigation Module Template
- Service Template
- Dashboard Widget Template

## Mallide eesmärk

- Vähendada arendusaega
- Tagada ühtne arhitektuur
- Vähendada vigade hulka
- Kiirendada prototüüpimist

## Template nõuded

- Järgib Cyber32 arhitektuuri
- Järgib Cyber32 liideseid
- Sisaldab dokumentatsiooni
- Sisaldab näidiskoodi

## Eesmärk

Võimaldada arendajatel alustada uut komponenti mõne minutiga.
