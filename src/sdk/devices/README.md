# SDK Devices

SDK Devices sisaldab juhiseid, malle ja tööriistu uute Cyber32 Device objektide loomiseks.

## Vastutus

- Device mallide pakkumine
- Device standardite kirjeldamine
- Device arendamise lihtsustamine
- Ühilduvuse tagamine Cyber32 süsteemiga

## Näited

- Sensor Device Template
- Motor Device Template
- Power Device Template
- Communication Device Template

## Device nõuded

- Võib kasutada mitut Driverit
- Kirjeldab oma Capabilities
- Omab Metadata objekti
- Järgib Cyber32 liideseid

## Device eesmärk

Ühendada üks või mitu Driverit terviklikuks kasutatavaks objektiks.

## Eesmärk

Võimaldada uute Device tüüpide loomist ühtse standardi järgi.
