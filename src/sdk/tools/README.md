# SDK Tools

SDK Tools sisaldab Cyber32 arendustööriistu.

Need tööriistad aitavad luua, testida, valideerida ja hallata Cyber32 komponente.

## Vastutus

- Arendustööriistade pakkumine
- Valideerimistööriistade pakkumine
- Testimistööriistade pakkumine
- Koodi genereerimise toetamine
- Dokumentatsiooni genereerimise toetamine

## Näited

- Module Generator
- Device Generator
- Driver Validator
- Capability Validator
- Documentation Generator
- Project Creator

## Tools eesmärk

- Kiirendada arendust
- Vähendada vigade hulka
- Tagada standardite järgimine
- Lihtsustada Cyber32 ökosüsteemi laiendamist

## Tools ei vastuta

- Süsteemi töö eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Runtime töö eest

Need ülesanded kuuluvad Cyber32 põhikomponentidele.

## Eesmärk

Muuta Cyber32 arendamine võimalikult lihtsaks ja ühtseks.
