# CAN

CAN HAL pakub ühtset ligipääsu CAN siinile.

## Vastutus

- CAN sõnumite saatmine
- CAN sõnumite vastuvõtmine
- CAN kontrolleri seadistamine
- Filtrite haldamine
- Riistvaraplatvormi detailide peitmine

## Näited

- Cyber32 moodulite vaheline suhtlus
- Robotite alamkontrollerid
- Akuhaldussüsteemid
- Tööstusseadmed
- Sensorivõrgud

## CAN ei vastuta

- Cyber32 protokolli eest
- Moodulite tuvastamise eest
- Andmete tõlgendamise eest
- Rakendusloogika eest

CAN HAL tegeleb ainult CAN siini tasemega.
