# SPI

SPI HAL pakub ühtset ligipääsu SPI siinile.

## Vastutus

- SPI seadmetega suhtlemine
- Andmete lugemine ja kirjutamine
- Seadmete valimine (Chip Select)
- Riistvaraplatvormi detailide peitmine

## Näited

- SD kaardid
- Ekraanid
- Flash mälud
- Ethernet moodulid
- Kiired andurid

## SPI ei vastuta

- Seadmete loogika eest
- Andmete tõlgendamise eest
- Plug-and-play registreerimise eest
- Rakendusloogika eest
