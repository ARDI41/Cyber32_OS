# I2C

I2C HAL pakub ühtset ligipääsu I2C siinile.

## Vastutus

- I2C seadmetega suhtlemine
- Andmete lugemine ja kirjutamine
- Seadmete adresseerimine
- Riistvaraplatvormi detailide peitmine

## Näited

- Temperatuuriandurid
- IMU andurid
- Ekraanid
- EEPROM kiibid
- Cyber32 moodulite tuvastus

## I2C ei vastuta

- Seadmete loogika eest
- Plug-and-play registreerimise eest
- Andmete tõlgendamise eest
- Rakendusloogika eest
