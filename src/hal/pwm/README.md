# PWM

PWM HAL pakub ühtset ligipääsu PWM signaalide genereerimiseks.

## Vastutus

- PWM signaali loomine
- Sageduse määramine
- Duty cycle määramine
- Riistvaraplatvormi detailide peitmine

## Näited

- Servo juhtimine
- Mootori kiiruse juhtimine
- LED heleduse juhtimine
- Helisignaali genereerimine

## PWM ei vastuta

- Servo loogika eest
- Mootori loogika eest
- Seadmete haldamise eest
- Rakendusloogika eest
