# UART

UART HAL pakub ühtset ligipääsu jadaliidesele.

## Vastutus

- Andmete saatmine
- Andmete vastuvõtmine
- Pordi seadistamine
- Baudrate haldamine
- Riistvaraplatvormi detailide peitmine

## Näited

- GPS moodulid
- LoRa moodulid
- Bluetooth moodulid
- Diagnostika ja debug
- Cyber32 moodulid

## UART ei vastuta

- Protokolli loogika eest
- Andmete tõlgendamise eest
- Plug-and-play registreerimise eest
- Rakendusloogika eest
