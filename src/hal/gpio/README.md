# GPIO

GPIO HAL pakub ühtset ligipääsu digitaalsetele sisenditele ja väljunditele.

## Vastutus

- Digitaalsisendite lugemine
- Digitaalväljundite juhtimine
- Pinni režiimi määramine
- Riistvaraplatvormi detailide peitmine

## Näited

- Nupp
- Lüliti
- Relee juhtimine
- LED juhtimine
- Digitaalne andur

## GPIO ei vastuta

- Servo juhtimise eest
- PWM genereerimise eest
- Analoogsignaalide lugemise eest
- Seadme loogika eest
