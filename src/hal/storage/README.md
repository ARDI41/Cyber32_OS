# Storage

Storage HAL pakub ühtset ligipääsu salvestusseadmetele.

## Vastutus

- Andmete lugemine
- Andmete kirjutamine
- Mälu oleku kontrollimine
- Salvestusseadmete abstraktsioon

## Näited

- Sisemine Flash
- EEPROM
- FRAM
- SD kaart
- Väline mälukiip

## Storage ei vastuta

- Konfiguratsioonide haldamise eest
- Failistruktuuri eest
- Andmete valideerimise eest
- Rakendusloogika eest

Need ülesanded kuuluvad kõrgematele kihtidele.
