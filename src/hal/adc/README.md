# ADC

ADC HAL pakub ühtset ligipääsu analoogsignaalide lugemiseks.

## Vastutus

- Analoogväärtuste lugemine
- Pinge mõõtmine
- ADC kanalite haldamine
- Riistvaraplatvormi detailide peitmine

## Näited

- Aku pinge mõõtmine
- Potentsiomeetri lugemine
- Analoogandurid
- Valgusandurid

## ADC ei vastuta

- Mõõtetulemuste tõlgendamise eest
- Andurite loogika eest
- Kalibreerimise eest
- Rakendusloogika eest
