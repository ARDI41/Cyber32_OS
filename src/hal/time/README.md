# Time

Time HAL pakub ühtset ligipääsu aja ja taimerite funktsioonidele.

## Vastutus

- Süsteemiaja lugemine
- Viivituste loomine
- Taimerite haldamine
- Perioodiliste sündmuste toetamine
- Riistvaraplatvormi detailide peitmine

## Näited

- Millisekundite loendur
- Sekunditaimer
- Watchdog tugi
- Ajastatud ülesanded
- Moodulite timeout kontroll

## Time ei vastuta

- Scheduleri loogika eest
- Rakenduse töövoogude eest
- Süsteemi olekute eest
- Kalenderfunktsioonide eest

Need ülesanded kuuluvad kõrgematele kihtidele.
