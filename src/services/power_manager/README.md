# Power Manager

Power Manager vastutab Cyber32 süsteemi energiakasutuse ja toitehalduse koordineerimise eest.

Power Manager teeb otsuseid energia kasutamise kohta, kasutades Power Device ja Power Module infot.

## Vastutus

- Toiteoleku jälgimine
- Energiakasutuse optimeerimine
- Aku seisundi jälgimine
- Energiapoliitikate rakendamine
- Madala energiatarbega režiimide haldamine

## Näited

- Aku tase alla 20%
- Madala energiatarbega režiimi aktiveerimine
- Laadimise jälgimine
- Kriitilise energiataseme tuvastamine
- Mitme toiteallika haldamine

## Power Manager võimaldab

- Hallata energiakasutust süsteemi tasemel
- Teavitada teisi komponente energiaseisundist
- Rakendada energiapoliitikaid
- Kaitsta süsteemi energiaga seotud vigade eest

## Power Manager ei vastuta

- Aku pinge mõõtmise eest
- BMS draiverite eest
- ADC lugemise eest
- Riistvara juhtimise eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
