# Security Manager

Security Manager vastutab Cyber32 süsteemi turvalisuse eest.

Security Manager tagab, et süsteem, moodulid, seadmed ja kasutajad järgivad Cyber32 turvareegleid.

## Vastutus

- Autentimine
- Autoriseerimine
- Moodulite usaldusväärsuse kontrollimine
- Turvasündmuste jälgimine
- Krüpteerimise toetamine
- Ligipääsuõiguste haldamine

## Näited

- Kasutaja autentimine
- Mooduli sertifikaadi kontroll
- Turvaline OTA uuendus
- API ligipääsu kontroll
- Krüpteeritud side

## Security Manager võimaldab

- Turvalist süsteemihaldust
- Moodulite usaldusväärsuse kontrolli
- Rollipõhiseid õigusi
- Turvalist andmesidet

## Security Manager ei vastuta

- Võrguühenduse loomise eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
