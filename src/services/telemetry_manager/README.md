# Telemetry Manager

Telemetry Manager vastutab Cyber32 süsteemi tööandmete kogumise, koondamise ja edastamise eest.

Telemeetria võimaldab jälgida süsteemi seisundit lokaalselt või kaugelt.

## Vastutus

- Telemeetria andmete kogumine
- Telemeetria andmete koondamine
- Telemeetria andmete edastamine
- Telemeetria sündmuste haldamine
- Telemeetria konfiguratsiooni haldamine

## Näited

- Aku tase
- CPU kasutus
- Mälu kasutus
- GPS asukoht
- Moodulite olek
- Võrguühenduse olek

## Telemetry Manager võimaldab

- Süsteemi jälgimist reaalajas
- Ajalooliste andmete kogumist
- Diagnostikat
- Kaugseiret

## Telemetry Manager ei vastuta

- Andurite lugemise eest
- Võrguühenduse loomise eest
- Riistvara juhtimise eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
