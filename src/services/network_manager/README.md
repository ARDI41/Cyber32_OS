# Network Manager

Network Manager vastutab Cyber32 süsteemi võrguühenduste haldamise eest.

Network Manager pakub ühtset liidest kõikidele võrguteenustele sõltumata kasutatavast tehnoloogiast.

## Vastutus

- Võrguühenduste haldamine
- Ühenduse oleku jälgimine
- Võrgukonfiguratsiooni haldamine
- Võrguteenuste koordineerimine
- Võrgu sündmuste vahendamine

## Näited

- WiFi
- Ethernet
- LoRa
- Bluetooth
- Internet Gateway

## Network Manager võimaldab

- Kontrollida ühenduse olekut
- Hallata võrguparameetreid
- Vahetada ühendustehnoloogiat
- Jagada võrguinfot teistele teenustele

## Network Manager ei vastuta

- Madala taseme side draiverite eest
- Riistvara juhtimise eest
- Plug-and-play eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.
