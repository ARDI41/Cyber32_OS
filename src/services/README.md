# Services

Services sisaldab Cyber32 süsteemi kõrgema taseme teenuseid.

Service kasutab Core, Registry, Runtime, Devices ja Modules kihte, et pakkuda süsteemile püsivaid funktsioone.

## Vastutus

- Süsteemiteenuste pakkumine
- Seadmete ja moodulite haldamine
- Võrgu, salvestuse, toite ja turvalisuse korraldamine
- Kõrgema taseme funktsioonide koordineerimine

## Alamkomponendid

- Device Manager
- Module Manager
- Network Manager
- Storage Manager
- Power Manager
- Telemetry Manager
- OTA Manager
- Security Manager

## Services ei vastuta

- Madala taseme riistvara juhtimise eest
- HAL teostuse eest
- Driverite sisemise töö eest
- UI renderdamise eest

Need ülesanded kuuluvad teistele kihtidele.
