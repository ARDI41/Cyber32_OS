# CAN Protocol

CAN Protocol kirjeldab Cyber32 CAN-põhist suhtlust.

CAN on eelistatud siini lahendus suurematele robotitele ja hajutatud süsteemidele.

## Vastutus

- CAN sõnumite standardiseerimine
- Moodulite vaheline suhtlus
- Telemeetria edastamine
- Käskude edastamine

## Näited

- Motor Controller
- Battery Module
- Sensor Hub
- Robot Arm Module

## Eelised

- Häirekindel
- Pikk kaablipikkus
- Mitme seadme tugi
- Tööstusstandard

## Seos Cyber32 Busiga

CAN on transpordikiht.

Cyber32 Bus määrab sõnumi sisu.
