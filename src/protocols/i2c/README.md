# I2C Protocol

I2C Protocol kirjeldab Cyber32 I2C-põhist suhtlust.

I2C on Cyber32 Plug-and-Play v0.1 peamine tuvastuskanal.

## Vastutus

- Metadata lugemine
- Moodulite tuvastamine
- Konfiguratsiooni lugemine
- Lihtsamate moodulite suhtlus

## Näited

- Sensor Module
- GPS Module
- Display Module
- EEPROM Module

## Eelised

- Lihtne
- Odav
- Vähe juhtmeid
- Lai tugi

## Seos Cyber32 Busiga

I2C on transpordikiht.

Cyber32 Bus määrab metadata vormingu.
