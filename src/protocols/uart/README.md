# UART Protocol

UART Protocol kirjeldab Cyber32 jadaliidese põhist suhtlust.

UART sobib seadmetele, millel on oma protsessor või keerulisem andmevahetus.

## Vastutus

- Jadaliidese suhtlus
- Diagnostika
- Moodulite andmevahetus
- Telemeetria

## Näited

- GPS
- LoRa
- Bluetooth
- Android Host

## Eelised

- Lihtne
- Kiire
- Lai riistvaratugi

## Seos Cyber32 Busiga

UART on transpordikiht.

Cyber32 Bus määrab sõnumi struktuuri.
