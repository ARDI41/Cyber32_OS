# Network Protocol

Network Protocol kirjeldab Cyber32 võrguühenduste põhist suhtlust.

See kiht võimaldab Cyber32 süsteemidel suhelda üle kohtvõrgu või interneti.

## Vastutus

- Kaugjuhtimine
- Telemeetria
- Pilveteenused
- Dashboard ühendused
- OTA uuendused

## Näited

- WiFi
- Ethernet
- VPN
- Cloud Gateway

## Eelised

- Piiramatu ulatus
- Kaugjuhtimine
- Mitme süsteemi ühendamine

## Seos Cyber32 Busiga

Network on transpordikiht.

Cyber32 Bus määrab sõnumi sisu ja tähenduse.
