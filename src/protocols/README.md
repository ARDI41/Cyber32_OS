# Protocols

Protocols sisaldab Cyber32 süsteemi suhtlusprotokolle.

Protokoll määrab, kuidas komponendid, seadmed ja moodulid omavahel andmeid vahetavad.

## Vastutus

- Andmevahetuse standardiseerimine
- Sõnumiformaatide kirjeldamine
- Ühilduvuse tagamine
- Cyber32 suhtlusreeglite defineerimine

## Alamkomponendid

- Cyber32 Bus
- CAN
- I2C
- UART
- Network

## Näited

- Mooduli identifitseerimine
- Capability vahetamine
- Telemeetria saatmine
- Käskude edastamine
- Mooduli registreerimine

## Protocols ei vastuta

- Riistvara juhtimise eest
- Driverite töö eest
- Plug-and-play eest
- Rakendusloogika eest

Need ülesanded kuuluvad teistele süsteemikomponentidele.

## Eesmärk

Tagada, et kõik Cyber32 komponendid räägiksid sama keelt.
