# Drivers

Drivers sisaldab konkreetsete seadmete ja komponentide juhtimise loogikat.

Driver kasutab HAL kihti ning pakub ülejäänud Cyber32 süsteemile ühtset viisi seadmete kasutamiseks.

## Vastutus

- Seadmete juhtimine
- Seadmetega suhtlemine
- Riistvaraspetsiifiliste detailide peitmine
- Andmete teisendamine Cyber32 jaoks sobivale kujule

## Driverite kategooriad

- Motors
- Sensors
- Actuators
- Displays
- Power
- Communication
- Storage

## Näited

- Servo Driver
- DC Motor Driver
- GPS Driver
- IMU Driver
- Relay Driver
- OLED Driver
- CAN Driver

## Drivers ei vastuta

- Plug-and-play tuvastuse eest
- Süsteemi registri eest
- Kasutajaliidese eest
- Rakendusloogika eest

Need ülesanded kuuluvad kõrgematele kihtidele.
