# Sensors

Sensors sisaldab anduritega seotud draivereid.

## Vastutus

- Andurite lugemine
- Andmete valideerimine
- Andmete teisendamine Cyber32 jaoks sobivale kujule
- Anduri konfiguratsioon

## Näited

- Distance Sensor Driver
- Temperature Sensor Driver
- Humidity Sensor Driver
- IMU Driver
- GPS Driver
- Light Sensor Driver
- Pressure Sensor Driver

## Sensors ei vastuta

- Navigatsiooni eest
- Otsuste tegemise eest
- Automaatika eest
- Kasutajaliidese eest

Need ülesanded kuuluvad kõrgematele kihtidele.
