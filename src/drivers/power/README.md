# Power

Power sisaldab toitehaldusega seotud draivereid.

## Vastutus

- Toiteallikate jälgimine
- Akuandmete lugemine
- Laadimisseadmete juhtimine
- Toiteoleku jälgimine
- Energiatarbimise mõõtmine

## Näited

- Battery Driver
- Charger Driver
- BMS Driver
- Power Monitor Driver
- DC-DC Converter Driver

## Power ei vastuta

- Energiapoliitika eest
- Toitehalduse otsuste eest
- Süsteemi töörežiimide eest
- Automaatika eest

Need ülesanded kuuluvad kõrgematele kihtidele.
