# Actuators

Actuators sisaldab täiturseadmetega seotud draivereid.

## Vastutus

- Täiturseadmete juhtimine
- Täiturseadmete oleku lugemine
- Täiturseadmete konfiguratsioon
- Ohutu töö tagamine

## Näited

- Relay Driver
- Solenoid Driver
- Valve Driver
- Buzzer Driver
- Lock Driver

## Actuators ei vastuta

- Automaatika loogika eest
- Otsuste tegemise eest
- Kasutajaliidese eest
- Süsteemi töövoogude eest

Need ülesanded kuuluvad kõrgematele kihtidele.
