# Displays

Displays sisaldab ekraanide ja visuaalsete väljunditega seotud draivereid.

## Vastutus

- Ekraanide juhtimine
- Teksti kuvamine
- Graafika kuvamine
- Ekraani konfiguratsioon

## Näited

- OLED Driver
- LCD Driver
- TFT Driver
- E-Paper Driver
- LED Matrix Driver

## Displays ei vastuta

- Kasutajaliidese loogika eest
- Rakenduse töövoogude eest
- Automaatika eest
- Süsteemi otsuste tegemise eest

Need ülesanded kuuluvad kõrgematele kihtidele.
