# Motors

Motors sisaldab mootoritega seotud draivereid.

## Vastutus

- Mootorite juhtimine
- Mootorite oleku lugemine
- Mootorite konfiguratsioon
- Mootorite turvaline käitamine

## Näited

- Servo Driver
- DC Motor Driver
- Stepper Driver
- BLDC Driver
- Linear Actuator Driver

## Motors ei vastuta

- Roboti liikumisloogika eest
- Navigatsiooni eest
- Automaatika eest
- Kasutajaliidese eest

Need ülesanded kuuluvad kõrgematele kihtidele.
