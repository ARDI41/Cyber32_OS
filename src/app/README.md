# App

App on Cyber32 süsteemi lõplik rakenduskiht.

Siin ühendatakse kõik Cyber32 komponendid töötavaks süsteemiks.

## Vastutus

- Süsteemi käivitamine
- Süsteemi konfiguratsioon
- Teenuste käivitamine
- Moodulite laadimine
- Süsteemi töövoo käivitamine

## Näited

- Cyber32 Core OS
- Planting Robot
- Rover
- Mower
- Home Automation Hub

## Eesmärk

Koostada Cyber32 komponentidest konkreetne töötav toode.

## App ei vastuta

- HAL teostuse eest
- Driverite teostuse eest
- Plug-and-play teostuse eest
- Dashboardi teostuse eest

Need ülesanded kuuluvad teistele kihtidele.
