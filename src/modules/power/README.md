# Power Modules

Power Modules sisaldab energia tootmise, salvestamise ja jaotamisega seotud mooduleid.

Need moodulid tagavad Cyber32 süsteemi tööks vajaliku energia.

## Vastutus

- Energiaallikate koondamine
- Toitevõimekuste kirjeldamine
- Energiaseisundi jälgimine
- Toitemooduli juhtimise lihtsustamine

## Näited

- Battery Module
- Power Supply Module
- Charger Module
- Solar Module
- Power Distribution Module

## Võimalikud Device objektid

- Battery Device
- Charger Device
- Power Monitor Device
- Storage Device
- Communication Device

## Power Module ei vastuta

- Süsteemi energiakasutuse poliitika eest
- Automaatika otsuste eest
- Rakendusloogika eest

Need ülesanded kuuluvad kõrgematele kihtidele.
