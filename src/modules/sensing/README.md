# Sensing Modules

Sensing Modules sisaldab keskkonna ja süsteemi jälgimisega seotud mooduleid.

Need moodulid koguvad andmeid ümbritsevast keskkonnast või Cyber32 süsteemist.

## Vastutus

- Andurite koondamine
- Mõõteandmete kogumine
- Andurivõimekuste kirjeldamine
- Andmete pakkumine teistele süsteemikomponentidele

## Näited

- Environment Module
- Weather Module
- Distance Sensing Module
- Soil Monitoring Module
- Safety Monitoring Module

## Võimalikud Device objektid

- Temperature Device
- Humidity Device
- Distance Sensor Device
- Pressure Sensor Device
- Light Sensor Device

## Sensing Module ei vastuta

- Otsuste tegemise eest
- Automaatika eest
- Mootorite juhtimise eest
- Kasutajaliidese eest

Need ülesanded kuuluvad kõrgematele kihtidele.
