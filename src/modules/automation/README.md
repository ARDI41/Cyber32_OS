# Automation Modules

Automation Modules sisaldab automaatika ja töövoogudega seotud mooduleid.

Need moodulid ühendavad seadmeid, andureid ja tegevusi automaatseteks protsessideks.

## Vastutus

- Automaatikavoogude koondamine
- Tingimuste ja tegevuste sidumine
- Automaatikavõimekuste kirjeldamine
- Korduvate protsesside lihtsustamine

## Näited

- Irrigation Module
- Greenhouse Module
- Home Automation Module
- Planting Automation Module
- Safety Automation Module

## Võimalikud Device objektid

- Sensor Device
- Relay Device
- Motor Device
- Power Device
- Communication Device

## Automation Module ei vastuta

- Madala taseme riistvara juhtimise eest
- Draiverite töö eest
- Süsteemi registri eest
- Kasutajaliidese eest

Need ülesanded kuuluvad teistele kihtidele.
