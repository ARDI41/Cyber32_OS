# Custom Modules

Custom Modules sisaldab kasutaja või arendaja loodud Cyber32 mooduleid.

Need moodulid võimaldavad süsteemi laiendada ilma Cyber32 põhikomponente muutmata.

## Vastutus

- Kohandatud moodulite koondamine
- Uute võimekuste lisamine
- Eksperimentaalsete lahenduste toetamine
- Projektipõhiste moodulite majutamine

## Näited

- Planting Robot Module
- Mower Module
- Rover Module
- Humanoid Module
- AI Assistant Module

## Võimalikud Device objektid

- Motor Device
- Sensor Device
- Communication Device
- Power Device
- Virtual Device

## Custom Module ei vastuta

- Cyber32 põhituuma muutmise eest
- HAL muutmise eest
- Driverite arhitektuuri muutmise eest

Need moodulid kasutavad olemasolevaid Cyber32 kihte.

## Eesmärk

Võimaldada uute lahenduste loomist ilma süsteemi põhistruktuuri rikkumata.
