# User Interface Modules

User Interface Modules sisaldab kasutajaga suhtlemiseks mõeldud mooduleid.

Need moodulid võimaldavad Cyber32 süsteemil kuvada infot, vastu võtta sisendeid ja pakkuda kasutajakogemust.

## Vastutus

- Kasutajaliidese komponentide koondamine
- Sisendite ja väljundite haldamine
- Kasutajaliidese võimekuste kirjeldamine
- Kasutajaga suhtlemise lihtsustamine

## Näited

- Display Module
- Touchscreen Module
- Control Panel Module
- Status Panel Module
- Voice Interface Module

## Võimalikud Device objektid

- Display Device
- Touch Device
- Button Device
- Buzzer Device
- Communication Device

## User Interface Module ei vastuta

- Äriloogika eest
- Automaatika otsuste eest
- Madala taseme riistvara juhtimise eest
- Süsteemi registri eest

Need ülesanded kuuluvad kõrgematele kihtidele.
