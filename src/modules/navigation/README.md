# Navigation Modules

Navigation Modules sisaldab asukoha, orientatsiooni ja liikumissuuna määramisega seotud mooduleid.

Need moodulid aitavad Cyber32 süsteemil mõista, kus ta asub ja kuhu liigub.

## Vastutus

- Asukoha määramine
- Suuna määramine
- Liikumise jälgimine
- Navigatsioonivõimekuste kirjeldamine

## Näited

- GPS Module
- RTK GPS Module
- IMU Module
- Compass Module
- Localization Module

## Võimalikud Device objektid

- GPS Device
- IMU Device
- Compass Device
- Storage Device
- Communication Device

## Navigation Module ei vastuta

- Mootorite juhtimise eest
- Teekonna planeerimise eest
- Automaatika otsuste eest

Need ülesanded kuuluvad kõrgematele kihtidele.
