# Motion Modules

Motion Modules sisaldab liikumisega seotud mooduleid.

Need moodulid võimaldavad Cyber32 süsteemil liikuda või liikumist tekitada.

## Vastutus

- Liikumisseadmete koondamine
- Liikumisvõimekuste kirjeldamine
- Liikumismooduli juhtimise lihtsustamine

## Näited

- Servo Module
- DC Motor Module
- Stepper Module
- BLDC Module
- Robot Drive Module

## Võimalikud Device objektid

- Servo Device
- Motor Device
- Encoder Device
- Power Device

## Motion Module ei vastuta

- Navigatsiooni eest
- Teekonna planeerimise eest
- Automaatika otsuste eest

Need ülesanded kuuluvad kõrgematele kihtidele.
