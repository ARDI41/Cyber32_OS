# Communication Modules

Communication Modules sisaldab andmeside ja ühenduvusega seotud mooduleid.

Need moodulid võimaldavad Cyber32 süsteemil suhelda teiste seadmete, moodulite ja võrkudega.

## Vastutus

- Sidevõimekuste koondamine
- Ühenduste haldamine
- Andmesidevõimekuste kirjeldamine
- Side kasutamise lihtsustamine

## Näited

- WiFi Module
- Bluetooth Module
- LoRa Module
- CAN Module
- Ethernet Module
- RS485 Module

## Võimalikud Device objektid

- WiFi Device
- Bluetooth Device
- LoRa Device
- CAN Device
- Ethernet Device

## Communication Module ei vastuta

- Rakendusprotokollide eest
- Automaatika otsuste eest
- Kasutajaliidese eest

Need ülesanded kuuluvad kõrgematele kihtidele.
